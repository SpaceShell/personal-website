What is this website about?

I tend to talk a lot about Pokémon. I ramble on about it so much to my friends. But, Not many of them know much about every Pokémon so this cool little website helps with that!

Alright, but what is a Pokémon?

A Pokémon is a creature that has a Pokédex number. Each Pokémon has a Pokédex number, which is a number that identifies them. Pokemon are known by many, and there's a diverse range of them. This website hopefully provides that clarification for newcomers!
